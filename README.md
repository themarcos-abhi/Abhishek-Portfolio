# Abhishek Kumar - Personal Portfolio

A modern, responsive, and interactive personal portfolio website built for Abhishek Kumar, a Full-Stack Developer and BCA student passionate about the MERN stack and Artificial Intelligence.

## 🚀 Features

*   **Modern Dark Theme:** Sleek design with a deep dark background (`#0a0a0a`), subtle gradients, and vibrant cyan/purple accents.
*   **Fully Responsive:** Fluid grids and CSS media queries ensure the site looks perfect on mobile, tablet, and desktop devices.
*   **Interactive Animations:** Smooth scroll animations, hover effects, and a dynamic hero background powered by Framer Motion.
*   **Experience & Certifications:** Dedicated section to showcase internships and professional certifications with links to view actual credentials.
*   **Projects Showcase:** Beautiful grid layout for displaying key projects with tech stack tags and links to live sites/source code.
*   **Contact Form:** Clean, functional contact section with integrated social links (GitHub, LinkedIn).
*   **Resume Integration:** Easy access to download or view the professional resume.

## 🛠️ Tech Stack

*   **Frontend Framework:** React 19
*   **Build Tool:** Vite
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS v4
*   **Animations:** Framer Motion
*   **Icons:** Lucide React

## 💻 Running Locally

To run this project on your local machine, follow these steps:

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your computer.

### Installation & Setup

1.  **Clone or Extract the Repository**
    If you downloaded the `.tar.gz` file, extract it to your desired location.

2.  **Navigate to the Project Directory**
    Open your terminal and navigate to the extracted folder:
    ```bash
    cd path/to/portfolio/directory
    ```

3.  **Install Dependencies**
    Run the following command to install all required packages:
    ```bash
    npm install
    ```

4.  **Start the Development Server**
    Launch the local Vite server:
    ```bash
    npm run dev
    ```

5.  **View in Browser**
    Open your browser and navigate to `http://localhost:5173` (or the URL provided in your terminal).
